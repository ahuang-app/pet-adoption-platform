## 项目概述

**暖心领养**是一个面向宠物爱好者的领养平台，提供宠物浏览、搜索筛选、用户注册、收藏、领养申请等功能。采用温暖治愈的视觉风格，目标是用技术连接等待领养的小动物和有爱心的领养者。

- **生产地址**: https://ahuang-app.github.io/pet-adoption-platform/
- **GitHub**: https://github.com/ahuang-app/pet-adoption-platform
- **Supabase 后端**: zsektdhyszbfawikgdyw.supabase.co

## 已完成功能

### 页面路由（7个）

| 路径 | 页面 | 说明 |
|------|------|------|
| `/` | 首页 | Hero → 搜索栏 → 宠物卡片 → 成功故事 → 收容所 → CTA |
| `/pets/:id` | 宠物详情 | 故事化叙事四段式布局 |
| `/search` | 搜索结果 | 筛选侧边栏 + 结果网格 |
| `/adopt/:petId` | 领养申请 | 宠物摘要 + 申请表单（需登录） |
| `/login` | 登录 | 邮箱/手机号双通道 |
| `/register` | 注册 | 邮箱/手机号双通道 |
| `/dashboard` | 用户中心 | 个人信息 + 收藏 + 申请记录 + 通知 |
| `/stories/:id` | 故事详情 | 成功领养故事详情页 |

### 核心功能

- **用户认证**: Supabase Auth 邮箱/手机号注册与登录，自动创建用户资料
- **宠物搜索**: 物种、体型、年龄范围筛选，Zustand 管理筛选状态
- **收藏功能**: 点击心形按钮切换收藏，Dashboard 查看收藏列表
- **领养申请**: 提交申请信息，自动记录到数据库，支持状态追踪
- **实时通知**: Supabase Realtime 订阅通知表，申请状态变更即时推送
- **头像上传**: Supabase Storage 上传头像，支持预览
- **个人信息**: 昵称修改，资料编辑

### 动效设计

- Hero 浮动装饰元素（emoji 动画）
- 宠物卡片 hover 上浮 + 图片缩放 + 暖色光晕
- 路由切换 fadeInUp
- 卡片网格交错入场动画
- 模态框弹性动画
- Notification badge 弹跳动画

## 数据库结构

| 表名 | 说明 | 记录数 |
|------|------|--------|
| species | 物种（狗/猫/兔） | 3 |
| breeds | 品种 | 11 |
| pets | 宠物信息 | 12 |
| users | 用户资料 | — |
| favorites | 收藏夹 | — |
| adoption_applications | 领养申请 | — |
| success_stories | 成功故事 | 6 |
| notifications | 通知 | — |

### RLS 安全策略

- 公开数据（pets/species/breeds/success_stories）：所有人可读
- 用户私有数据（users/favorites/applications/notifications）：仅本人可读写
- Storage（avatars bucket）：认证用户可上传，所有人可查看
